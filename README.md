# Anox
Linux Digital Footprints Remover
<img src="https://github.com/Trabbit0ne/Anox/blob/main/medias/preview.gif" style="width: 75%;">

## Installation
```
git clone https://github.com/Trabbit0ne/Anox && cd Anox && chmod +x *
```
Then
### Remover
```
./main.sh
```
OR
### Detector
```
./detector.sh
```
